ALICE (All-In-one Consultant Agent)
정부 R&D 사업계획서 작성을 위한 AI 기반 초격차 컨설팅 에이전트 

ALICE는 복잡한 정부 공고문과 양식을 Vision API와 RAG 기술로 정밀 분석하여, 연구자가 행정 업무 대신 연구 본연의 가치에 집중할 수 있도록 돕는 AI 에이전트 서비스입니다
